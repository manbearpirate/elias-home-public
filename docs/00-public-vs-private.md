# Public vs Private: What goes where

## Public repo includes
- Roadmap, feature matrix, guardrails (high level)
- Conceptual ADRs (decisions without household specifics)
- High-level architecture

## Private repo includes
- Any household-specific details:
  - network topology, IPs, device IDs, Wi-Fi info
  - physical device locations
  - automation endpoints, tokens, secrets
  - detailed implementation/runbooks
  - specific notification targets and personal schedules
